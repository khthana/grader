# 05 · Teacher — รายชื่อนักศึกษา (CRUD + นำเข้า Excel + แบ่งหน้า)

> อ่าน `00-foundation.md` ก่อน · ภาพ: `../screens/teacher-students.png` · บทบาท: **Teacher**

## Purpose
จัดการรายชื่อนักศึกษาในรายวิชา — เพิ่ม/แก้ไข/ลบ, นำเข้าจากไฟล์ Excel, ค้นหา, กรองกลุ่มเรียน, แบ่งหน้า

## Layout
- **PageHead:** "รายชื่อนักศึกษา" + คำบรรยาย "{รหัสวิชา} · {ชื่อวิชา}" + actions: ปุ่ม **"นำเข้า Excel"** (outline, icon upload) + **"เพิ่มนักศึกษา"** (primary, icon plus)
- **การ์ด** บรรจุ toolbar + ตาราง + pagination

### Toolbar
- ซ้าย: ช่องค้นหา (ค้นรหัส / ชื่อ / นามสกุล)
- ขวา: select **กลุ่มเรียน** (ทุกกลุ่ม / กลุ่ม 1 / กลุ่ม 2) + ปุ่ม **"ส่งออก"** (ghost sm, icon download)

### Table
| คอลัมน์ | รายละเอียด |
|---|---|
| # | ลำดับต่อเนื่องตามหน้า |
| รหัสนักศึกษา | mono |
| คำนำหน้า | นาย/นางสาว/นาง |
| ชื่อ - นามสกุล | หนา |
| หลักสูตร | muted (เช่น วิศวกรรมคอมพิวเตอร์) |
| กลุ่ม | Badge |
| ปีการศึกษา | muted |
| (actions) | IconBtn แก้ไข + ลบ(danger) |

- ถ้าค้นไม่พบ → แถว empty "ไม่พบนักศึกษาที่ค้นหา"

### Pagination
- **10 แถว/หน้า** · มุมขวาล่าง "X จาก Y รายการ" + ปุ่มหน้าแรก/ก่อนหน้า/เลขหน้า(มี …)/ถัดไป/หน้าสุดท้าย
- ค้นหา/เปลี่ยน filter → รีเซ็ตกลับหน้า 1

## Behavior

### Modal เพิ่ม/แก้ไขนักศึกษา
ฟิลด์ (กริด 2 คอลัมน์): รหัสนักศึกษา* · คำนำหน้า(select) · ชื่อ* · นามสกุล* · กลุ่มเรียน(select) · ปีการศึกษา · footer "ยกเลิก" / "บันทึก"

### Modal ลบ
ยืนยัน "ต้องการลบ **ชื่อ** (รหัส) ออกจากรายวิชาใช่หรือไม่?" + "ยกเลิก" / "ลบ"(danger)

### Modal นำเข้า Excel (width ~620)
- **Dropzone** ลากวาง: ค่าปกติแสดงไอคอน upload + "ลากไฟล์ .xlsx มาวาง หรือคลิกเพื่อเลือก" + hint คอลัมน์ที่รองรับ (รหัส, คำนำหน้า, ชื่อ, นามสกุล)
  - สถานะ drag-over: เส้นขอบ+พื้น `--blue`
  - เมื่อมีไฟล์: เส้นขอบ+พื้น **เขียว**, ไอคอน doc, ชื่อไฟล์, "พบ N รายชื่อ พร้อมนำเข้า"
- ลิงก์ **"ดาวน์โหลดเทมเพลตไฟล์ตัวอย่าง"** (icon download)
- เมื่อมีไฟล์ → แสดง **ตารางพรีวิว** ข้อมูลที่จะนำเข้า (รหัส/คำนำหน้า/ชื่อ/นามสกุล)
- footer: "ยกเลิก" / **"นำเข้า (N รายชื่อ)"** (disabled จนกว่าจะมีไฟล์)
- ของจริง: parse .xlsx/.csv (เช่น SheetJS) ฝั่ง client หรือส่ง backend; validate รหัสซ้ำ/รูปแบบก่อนยืนยัน
