# 09 · Student — สมุดคะแนน (ของตนเอง)

> อ่าน `00-foundation.md` ก่อน · ภาพ: `../screens/student-scores.png` · บทบาท: **Student**

## Purpose
แสดงคะแนน **เฉพาะของนักศึกษาเอง** (ห้ามเห็นของคนอื่น)

## Layout
- **PageHead:** "สมุดคะแนนของฉัน" + คำบรรยาย "{รหัสนักศึกษา} · {ชื่อ}"
- **Banner สรุป** (การ์ดแนวนอน):
  - **วงแหวน progress** (SVG donut) แสดง % คะแนนรวม (วงในตัวเลข %)
  - ข้างขวา: คะแนนรวม "X / Y คะแนน" (ตัวใหญ่) + "ทำได้ a จาก b โจทย์"
- **การ์ดตาราง**

### Table
| คอลัมน์ | รายละเอียด |
|---|---|
| โจทย์ | ลำดับ |
| ชื่อโจทย์ | หนา |
| สถานะ | Badge — ตรวจแล้ว=เขียว / ยังไม่ส่ง=เหลือง |
| คะแนนที่ได้ | ตัวเลขเน้น (เขียว) หรือ "–" ถ้ายังไม่ส่ง |
| เต็ม | muted |

- แถวล่างสุด = **แถวรวม** (คะแนนรวม / เต็มรวม, พื้น `--blue-50`)
- **⚠️ ไม่มีคอลัมน์ "ระดับ/Level"**

## Behavior
- อ่านอย่างเดียว — ไม่มี action แก้ไข
- ของจริง: ดึงเฉพาะคะแนนของ user ที่ล็อกอิน (scoping ฝั่ง backend)

## Data
ใช้คะแนนของนักศึกษาคนเดียว (ใน prototype = แถวแรกของ SCOREBOOK) เทียบกับรายการ **Problem** เพื่อหาคะแนนเต็มต่อโจทย์; % = คะแนนรวม ÷ เต็มรวม
